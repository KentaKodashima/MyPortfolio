Amtrip

Category:

iOS app

LANGUAGE:

Swift 4

FEATURES:

Realm, Google Places Autocomplete API

Production:

March 2018 - July 2018

Top Page

  1. When there is no data, it shows texts saying "There are no pages yet.".
  2. The background picture is randomly piccked from the images of the pages which belong to the album.
  3. The date label would be updated when the user creates the new page which has older date.
  4. I chose UICollecUICollectionView to show thumbnails because it's more flexible than UITableView.

Creation Page

  1. When the user taps on "Album Title" textField, AlbumCreateVC shows up. This let user choose from existing album or create a new album.
  2. When the user taps on "Date" textField, UIDatePicker shows up. Texts change responding to the user's action. The default value is today's date.
  3. When the user taps on "Location" textField, Autocomplete Controller from Google Places API shows up. The user can easily search and pick a location.
  4. Tapping on the buttons with the camera icon and library icon brings up UIImagePickerController.
  5. In order to display thumbnails of images, I used UICollectionView horizontally.
  6. The images is copied and stored into app's FileManager directory. Database just knows the path to the images.
  7. An image is deleted from UI and FileManager directory by tapping the delete button at the right above of UICollectionViewCell.

Favorite Page

  1. Only shows pages which selected as a favorite page.
  2. Toggling the favorite state with the barButtonItem on detail page.

Seach Page

  1. Its UI show pages which contain texts on UISearchView.
  2. Filtering the RealmObjects using NSPredicate with searchText.

Your Page

  1. Using UISegmentedControl to show different contents.
