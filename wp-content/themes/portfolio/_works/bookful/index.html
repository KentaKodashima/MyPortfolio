App Summary

Bookful is a simple tool to keep records of the books you've read.
It is kind of a hard task to remember the contents of the books by reading just one time.
Making reading records using this app might help you remember.
This app is available on the Google Play Store.

Top Page

  1. When there is no data available, it shows texts saying "There is no record yet.".
  2. The cells are listed using RecyclerView.
  3. In order to show TextView at the bottom of a cell, I put a gradient View on the thumbnail image.
  4. In order to make margins around RecyclerView items, I created the CustomItemDecoration class.

Creation Page

  1. The gray default image is replaced when the user uploads an image.
  2. The image is copied into file directory and stored as an String image path by RealmObject.
  3. Record object is created only when the user taps the "SAVE THIS PAGE" button at the bottom.
  4. The user can leave the image empty.
  5. If the user has empty fields, AlertDialog shows up.

Search Page

  1. Its UI shows records which contain texts from the search box in Records' title.
  2. The cells are listed using RecyclerView.
  3. The cells can be swipe to delete.
  4. Filtering the RealmObjects using onQueryTextChange() with newText from the search box.
  5. When there is no data available, it shows texts saying "There is no record yet.".

Record Detail

  1. It displays the data from the Record object which has the specific recordKey.
  2. When the user taps the button at the right top of action bar, 'Edit' and 'Delete' menu appears.
  3. 'Edit' on the menu leads to Editing activity.
  4. 'Delete' on the menu actually deletes the Record from database and leads to the previous activity.

Edit Record

  1. The user can edit any information related to the Record.
  2. Instead of putting "SAVE THIS PAGE" button at the bottom, I put "Done" button at the right top of the action bar.
